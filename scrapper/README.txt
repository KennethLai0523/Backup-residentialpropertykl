# Mudah → IPK Listing Helper

## Install
1. Open Chrome: `chrome://extensions`
2. Turn on **Developer mode**
3. Click **Load unpacked**
4. Select this folder: `mudah_ipk_extension`

## Use
1. Open Mudah listing page
2. Click extension → **1. Listing Extract**
3. Open photo gallery popup
4. Click extension → **2. Photo Extract**
5. Open your `admin.html`
6. Click extension → **3. Listing Paste**
7. Click extension → **4. Photo Paste / Copy URLs**

Note: Chrome cannot auto-fill `<input type="file">` from remote URLs for security. The photo paste button copies/shows image URLs. To fully automate photo upload, your admin page needs image URL fields or a backend downloader.
